# 🚀 РУКОВОДСТВО ПО ЗАПУСКУ — EMA Paper Trading Bot

## ЧТО ДЕЛАЕТ БОТ
- Следит за BTCUSDT в реальном времени (данные с Binance, бесплатно)
- Торгует по EMA(21/50) + ATR стратегии
- Виртуальный баланс $10 000 — реальных денег НЕ тратит
- Записывает каждую сделку в SQLite
- Работает 24/7

## 📦 УСТАНОВКА (Ubuntu / Oracle Cloud)

```bash
# 1. Подключись по SSH к серверу
ssh ubuntu@ВАШ_IP

# 2. Установи Python и зависимости
sudo apt update && sudo apt install -y python3 python3-pip screen

# 3. Скопируй файлы бота на сервер
# Вариант A: через scp (с локального ПК)
scp -r trading_bot/ ubuntu@ВАШ_IP:~/

# Вариант B: создай файлы вручную на сервере через nano

# 4. Перейди в папку и установи зависимости
cd ~/trading_bot
pip3 install -r requirements.txt

# 5. Запусти в фоне через screen (бот не остановится при закрытии SSH)
screen -S trading_bot
python3 bot.py

# Отключиться от screen (бот продолжает работать):
# Нажми Ctrl+A, затем D

# Вернуться к логам бота:
screen -r trading_bot
```

## 📅 ПЛАН НА МЕСЯЦ

| Неделя | Действие |
|--------|----------|
| 1      | Запустить бот, проверить логи, убедиться что сделки открываются |
| 2      | Смотреть каждый вечер: `python report.py` |
| 3      | Если 0 сделок → проверить ATR-фильтр (возможно флэт на рынке) |
| 4      | Итоговый отчёт: Win Rate, PF, Expectancy, Drawdown |
| Вывод  | PF > 1.2 + 30+ сделок → пробовать на Bybit Testnet с реальными ордерами |
